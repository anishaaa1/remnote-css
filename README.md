<!-- Rainbow Headers -->
## 🔌 Rainbow Headers
```css
@import url("https://anishaaa1.github.io/remnote-css/css/rainbow-headers.css");
```

<details>
<summary>Notes</summary>

* Create these tags:
  * `H4` for Header 4
  * `H5` for Header 5
  * `H6` for Header 6

> ❗️Important as they are not native headers unlike Header 1, Header 2, and Header 3.

</details>
